# Cloak-android
Shadowsocks plugin Cloak for Android

## Requirements

- Go 1.15
- Android NDK 21.3.6528147
- Android SDK 29
- Gradle 6.1.1

## Build Instructions

1. Install the appropriate SDK & NDK versions in Android Studio's SDK manager
2. Execute `cd app/src && ./make.sh`
3. Build with Android Studio
